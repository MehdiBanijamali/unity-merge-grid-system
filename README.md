This repository contains key sample scripts from a gameplay-focused merge & rebuild island system implemented in Unity (C#). The scripts demonstrate modular design, scalable architecture,
and core gameplay logic for a merge-style game where players rebuild an island by combining items.

Features Demonstrated
Merge System Architecture: Object merging logic and grid handling concepts.
Task / Quest System: Sample task management and event-driven gameplay structure.
Modular, Scalable Design: Scripts are organized for maintainability and future expansion.

Key Systems & Patterns:
Grid & Merge Management: Handles item positioning and merge logic.
Event-Driven Architecture: Observer pattern for gameplay events.
Factory & Singleton Patterns: Dynamic object creation and manager classes.
Included Sample Scripts

The repository includes three core modules demonstrating system architecture:

Grid System – Core grid logic for item placement and merge handling.
Merge Logic – Item combination and progression logic.
Task / Order System – Sample system for managing objectives and game events.

These scripts are standalone and can be imported into a Unity project to explore and expand.
